SpeedyLink - Options

SpeedyLink allows you to quickly navigate to predefined URLs with customizable input parameters at the end. It is useful for JIRA and other websites.

Example:

  • Insert 'Marketing' for the name and 'https://jira.demo.almworks.com/projects/MKT/issues/MKT-' for the URL.

  • Open the popup by clicking the icon in the upper right corner (you can also use the hotkey Ctrl + Shift + Alt + S).

  • Type the number '3' and press [Enter].

The page https://jira.demo.almworks.com/projects/MKT/issues/MKT-3 will be opened.
